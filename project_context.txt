# 📘 FES Mentora Backend (RAG + Streaming Chat)

This repository contains the backend for **Mentora**, an AI-powered virtual study-abroad counsellor for **FES**.
The system is built using **FastAPI**, **RAG (Pinecone)**, and **OpenAI**, with support for **real-time streaming**, **contact fast-paths**, and **memory-aware conversations**.

---

## 🚀 How to Run

From the **project root**, run:

```bash
python -m uvicorn app.main:app --reload
```

Server will start on:

```
http://127.0.0.1:8000
```

## 🗂️ Project Structure (High-Level)

```
app/
├── core/          # App-wide configuration, middleware, security
├── models/        # Pydantic request/response schemas
├── routers/       # API route definitions
├── services/      # Business logic (LLM, RAG, vector DB)
├── utils/         # Small helpers and fast-path logic
└── main.py        # Application entry point
```

Each folder has a **single responsibility**.

## 🧠 Core System Behavior (Important)

### 🔁 Query Flow (Simplified)

```
Incoming Request
   ↓
Is contact query?
   ├─ Yes → contact_utils (fast, deterministic)
   └─ No
        ↓
Conversation memory (short-term)
        ↓
RAG (Pinecone)
        ↓
OpenAI response (streaming or non-streaming)
```

---

### 🧠 Memory Design

* **Short-term conversation memory**

  * Keeps last few user/assistant turns
  * Enables follow-up questions
  * Stored in memory (per session)

* **No LLM-based “self memory”**

  * All memory behavior is explicit and deterministic
  * Prevents hallucination and instability
